
### AURYN TWEAKING
1. Download Auryn 
2. Remplace auryn/src file with the src file inside Auryn_src
3. Follow instruction from auryn readme for the installation

### SRC TWEAKING

1. In the src directory we will need to modify globalvars.sh 
2. Change the OUTDIR_ROOT at your convenience. It will indicate where simulations datas are stored 
3. Change the python path at your convenience.

### ANALYSE TWEAKING

1. In the ana folder open the notebook
2. Change the auryn location to import the python tools from auryn (first cell)
3. Change the datadir to match tour simulation directory
4. install all the necessary libraries :

Numpy, matplotlib,Pandas,scipy,seaborn,numba

Should be all ;)
